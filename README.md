## Cypress + SvelteKit bug

This repo provides a minimal reproduction of a bug where building Cypress tests with vite (using the [cypress-vite](https://github.com/mammadataei/cypress-vite) plugin) fails.

To reproduce:

```sh
$ yarn
$ yarn cypress
```
